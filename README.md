# Trabajo Práctico Nº4 - Inteligencia Artificial

**Alumno:** Luciana Liuzzi

## Descripción

Este repositorio contiene el desarrollo del Trabajo Práctico Nº4 de la materia Inteligencia Artificial.

### Contenido

- Implementación de la Transformada de Hough para rectas.
- Implementación de la Transformada de Hough para circunferencias.
- Ejemplos de prueba.
- Resultados obtenidos.

El objetivo del trabajo es analizar el uso de la Transformada de Hough para resolver el problema de detección de elementos geométricos en una línea automatizada de montaje de motores.
